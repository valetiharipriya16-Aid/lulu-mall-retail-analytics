# Lulu Mall Retail Analytics Dashboard

## Project Overview
Developed an interactive Excel dashboard to analyze 12 months of retail sales data across Lulu Mall stores in India.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- VLOOKUP
- INDEX-MATCH
- Slicers
- Conditional Formatting

## Dashboard Features
- Total Revenue KPI
- Total Orders KPI
- Total Quantity KPI
- Total Customers KPI
- Revenue by Month
- Revenue by Store
- Revenue by Product Category
- Revenue by Gender
- Payment Mode Distribution
- Interactive Slicers
